# KBLI 2020 Interactive Mind Map

Repositori ini berisi visualisasi interaktif berbasis HTML untuk menelusuri struktur **KBLI 2020** dalam bentuk mind map untuk membantu pembacaan kode dan hierarki KBLI secara lebih cepat, terutama ketika digunakan untuk keperluan pemetaan kegiatan usaha, analisis perizinan, atau pendampingan penyusunan matriks pemanfaatan ruang.

## Fitur

- Tampilan mind map interaktif berbasis HTML
- Mode expand dan collapse untuk membuka atau menutup seluruh cabang
- Fitur pencarian kode dan nama kegiatan
- Tombol reset untuk mengembalikan tampilan awal
- Struktur hierarkis dari kategori utama sampai kode KBLI 5 digit

## Catatan Regulasi

KBLI 2020 diatur berdasarkan **Peraturan BPS Nomor 2 Tahun 2020**. Saat ini, KBLI telah memasuki masa transisi menuju **KBLI 2025** yang ditetapkan melalui **Peraturan BPS Nomor 7 Tahun 2025**.

Berdasarkan informasi BPS, implementasi KBLI 2025 secara nasional dijadwalkan paling lambat **18 Juni 2026**. Selama masa transisi, KBLI 2020 dan KBLI 2025 digunakan secara paralel sampai proses penyesuaian selesai.

Pelaku usaha perlu memeriksa kesesuaian kode aktivitas usahanya dengan KBLI 2025 melalui sistem **OSS RBA** atau sistem terkait lainnya. Jika terdapat perubahan substansi, seperti perubahan maksud, tujuan, atau ruang lingkup kegiatan usaha, pelaku usaha perlu melakukan penyesuaian melalui OSS/AHU. Jika tidak ada perubahan substansi, penyesuaian atau konversi dapat dilakukan secara otomatis oleh sistem sesuai ketentuan yang berlaku.

## Batas Waktu Transisi

Batas waktu penyesuaian penggunaan KBLI 2025 adalah:

**18 Juni 2026**

Tanggal ini merujuk pada masa penyesuaian paling lambat 6 bulan sejak Peraturan BPS Nomor 7 Tahun 2025 diundangkan.

## Kegunaan

File ini dapat digunakan untuk:

- Membantu membaca struktur KBLI 2020 secara visual
- Menelusuri kode kegiatan usaha berdasarkan nama atau kode
- Mendukung diskusi teknis terkait perizinan berusaha
- Membantu penyusunan matriks kegiatan usaha, termasuk untuk kebutuhan analisis tata ruang
- Menjadi alat bantu sementara selama masa transisi KBLI 2020 menuju KBLI 2025
